# front

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
## Для запуска на проде

docker-compose up -d --build

## Для запуска на dev

* `npm i`
* `npm run dev`
